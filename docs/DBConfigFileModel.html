Configuration by DBConfig File descriptor

Property Keys Description
db_type: mysql; postgresql; firebirdsql; sqlserver; Determines the target RDBMS for connect
host: localhost; any valid server IP; Determines the address of database server
port: 3306; 5405; etc; Determines the port of database server
user: root; sa; SYSDBA; SYSTEM; etc; Determines the user of database server
password: yourpasswordkey Determines the password of database server
database: sales; peoples; products; customers; etc; Determines the default database to connect in server

Initialization example:
SessionFactory sf = Configuration.createSessionFactory("mysql");

Content file example:

-- Database config --

db_type:mysql
host:localhost
port:3306
user:root
password:81547686
database:cadastro

-- Connection pool config --

acquire_increment:1
max_statements:5
max_pool_size:40
min_pool_size:3

-- Miscelania --

persistence_context:com.mycompany.myproject.MyContextClass
persistence_logger:com.mycompany.myproject.MyLoggerClass
